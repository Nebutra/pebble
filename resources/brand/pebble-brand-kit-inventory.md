# Pebble / 溪石 Brand Kit Inventory

This document maps the current brand assets to the Pebble replacement set. It
is intentionally scoped to brand kit assets, not the application UI system.

Brand naming:

- English product name: `Pebble`.
- Chinese product name: `溪石`.
- Stable technical identifiers remain English (`pebble`, `PEBBLE_*`,
  package names, URLs, and repo paths).

## Migration Order

1. Generate reference-locked brand assets into `resources/brand/generated/`
   using the approved no-background `>_` pebble direction.
2. Use `resources/brand/approved/pebble-logo.png` as the approved
   primary mark reference.
3. Generate the production icon family from the selected mark only.
4. Hand-vectorize the selected mark into a clean SVG. The generated image is a
   design reference, not the final production vector.
5. Replace source SVGs and React Native vector component.
6. Run `pnpm build:icons` to regenerate ICNS/ICO/PNG outputs.
7. Resize and crop mobile/docs assets from the selected masters.
8. Validate with image inspection at 1024, 256, 48, and dark/light backgrounds.
9. Run focused icon tests and packaging asset checks.

## Generation DAG

The first generation pass is deliberately serial where identity consistency
matters:

| Stage | Output | Why it exists |
|---|---|---|
| Foundation | `foundation-material-board` | Locks palette, matte pebble material, paper grain, and light direction. |
| Semantics | `foundation-pebbling-symbols` | Turns the pebbling philosophy into reusable visual motifs. |
| Mark seed | `core-mark-seed` | Combines material and terminal prompt semantics into a first single-pebble mark. |
| Mark cleanup | `core-mark-production-clean` | Removes clutter before the mark is used downstream. |
| App anchor | `app-icon-classic` | Establishes the transparent freestanding desktop icon family anchor. |
| Variants | `app-icon-soft-watercolor`, `app-icon-technical-mineral` | Preserve the same mark while changing mood/material. |
| Production proof | `monochrome-glyph-chromakey`, `small-icon-legibility-proof` | Supports alpha extraction, vectorization, and tiny-size review. |
| Public materials | `mobile-splash-mark`, `readme-hero-background`, `onboarding-soft-pattern`, `social-card-background` | Reuse the same icon/material/symbol anchors. |
| Overview | `brand-system-overview` | Joins the family for visual consistency review. |
